# 自动预约系统

本脚本针对Fudan的场馆预约，逆向了场馆预约前端，实现了无需浏览器即可预约场馆，空出计算机使用。

## 设置

本脚本给出了`config.json.example`，将其改为`config.json`并填入对应信息即可开始预约。

## 运行
安装依赖后，运行命令
```
python auto.py
```
即可开始等待预约。
